# Calm and Courageous Sleepers

Static Astro landing page for Calm and Courageous Sleepers.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production build is generated in `dist/`.

## AWS Amplify deployment

1. Push this project to GitHub.
2. Open AWS Amplify.
3. Choose **Host web app**.
4. Connect the GitHub repo.
5. Amplify will use `amplify.yml` automatically.
6. Add the custom domain `calmandcourageoussleepers.com` in Amplify domain settings.
